# Create Cluster

## Set Env Vars
 First export the following env vars (DO_TOKEN, DO_ACCESS_KEY, DO_SECRET_KEY). As per below
 ```
 export DO_TOKEN="<VALUE>" && export DO_ACCESS_KEY="<VALUE>" && export DO_SECRET_KEY="<VALUE>"
 ```

## Create Cluster
```
make
```
